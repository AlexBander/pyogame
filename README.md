# OGame



## Installation

`pip install ogame`



## Usage

>>> import OGame
>>> ogame = OGame(USER, PASW)
>>> ogame.check_attack()
>>> ogame.build(OGame.CRYSTAL_MINE)
>>> ogame.build(OGame.SOLAR_SAT)
>>> ogame.
